# Gulp Starter
(In Progress) starter gulpfile.js file for frontend that can compile and live reload when less, sass, js, html are changed.

## Install
- `npm install -g gulp`
- `npm install --save-dev gulp gulp-connect gulp-less gulp-sass`

## How to use
`gulp` or `gulp default`
